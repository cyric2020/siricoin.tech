A short story on the birth of Siricoin

This is how our coin started

SiriCoin first started as a token on polygon network which was a mineable token, at the time we only had webminer and there was no support to gpu/avr mining. Suddenly there was a huge bug that caused the network to crash and the users werent able to mine. So we tried a lot to fix the bug but it was impossible, since then we began to create our own mainnet/network for siricoin. After this we upgraded the old website to the new and clean look website. From there our journey of Siricoin began, continuing without and its still going without any issues...

What is SiriCoin?
SiriCoin is a lightweight crypto which can be mined through low-powered devices like CPU's, GPU's microcontroller's such as Arduinos and ESP's, RPI's and Mobile devices.
